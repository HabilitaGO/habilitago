# LGPD Compliance — HabilitaGO

## Fundamento Legal

HabilitaGO opera em conformidade com a **Lei Geral de Proteção de Dados (Lei nº 13.709/2018 — LGPD)**.

## Princípios Aplicados

| Princípio LGPD | Como Aplicamos |
|---------------|---------------|
| **Finalidade** | Dados coletados apenas para operação da plataforma e obrigações legais |
| **Adequação** | Coleta proporcional à finalidade declarada |
| **Necessidade** | Mínimo de dados necessários para cada funcionalidade |
| **Livre Acesso** | Usuários podem consultar seus dados a qualquer momento |
| **Qualidade** | Dados mantidos atualizados e precisos |
| **Transparência** | Política de privacidade pública em `/privacidade` |
| **Segurança** | Criptografia em repouso e em trânsito, controles de acesso |
| **Prevenção** | Medidas proativas para evitar danos |
| **Não Discriminação** | Dados não usados para fins discriminatórios |
| **Responsabilização** | Logs de auditoria, trilhas de evidência |

## Dados Coletados

### Alunos
- Nome, e-mail, telefone (cadastro e comunicação)
- Histórico de aulas e pagamentos (contrato de serviço)
- Avaliações dadas (publicadas com nome)
- Localização aproximada (busca de instrutores próximos)

### Instrutores
- Nome, e-mail, telefone, CPF/CNPJ
- CNH especial, certificados DETRAN (validação de credenciais)
- Foto de perfil pública
- Dados bancários (repasse via Stripe Connect)

### Dados Técnicos
- Logs de acesso e autenticação (segurança e auditoria)
- Cookies essenciais (sessão e segurança)
- Cookies analíticos e de marketing (mediante consentimento)

## Retenção de Dados

- **Dados de conta ativa:** mantidos durante o período de uso
- **Dados financeiros:** retidos pelo período legal mínimo (5 anos)
- **Logs de segurança:** retidos por 12 meses
- **Dados pós-exclusão:** anonimizados ou deletados conforme política interna
